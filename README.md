# 哔哩哔哩歌词弹幕姬
歌词弹幕输入辅助 Chrome 插件


## 基本功能
1. 解析 LRC 歌词数据，根据数据内容自动调整进度条并展示当前的歌词，便于输入歌词弹幕。
2. -10秒~+10秒的歌词调整功能。

## 接下来实现的功能
1. 前端细节修复，风格统一。
2. 自动切换到 H5 播放器。
3. 歌词进度跟随进度条。
4. 解析在线歌词数据。

## 存在的问题
1. 无法解析多时间轨的 LRC 数据。
2. 代码存在优化空间。
